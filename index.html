    MANUAL ROBOTICS

    MANUAL ROBOTICS is where the participant needs to control his/her own bot via a remote. The bot can't go on it's own. This is an event where the participant not only needs to make a great design but also needs to have great proficiency in the control mechanism of the bot so as to pass all the hurdles and take away the prize

    We have a wide range of events for MANUAL ROBOTICS , ranging from a bot which has to pass the real world terrains to a bot where it has to play soccer.



    AUTONOMOUS ROBOTS

    Autonomous Robot deals with those robots which does not need any user control. All you need to do is program your bot and leave the rest to itself.

    We have three different events for autonomous robots, starting from Blackolyte which is a line follower event, where the bot needs to follow a line of particular colour and reach the finish line. The second event which is Shield Bot is an event where the bot needs to detect and push objects from a rotating disk.

    XANT

    This is a call for all the coders around us. We all know what we do over here. Three primary things we need to do here, first is we code, second is we code again and last is we code all over again.

    BRIDGE BUILDING

    Bridge Building is where engineers gets to show their learning and how they will be changing India with greater connectivity in future.
